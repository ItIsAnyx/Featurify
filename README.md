\# Featurify



\*\*Featurify\*\* - AI-ассистент для анализа датасетов и генерации рекомендаций по Feature Engineering.



Цель приложения - ускорить подготовку данных и сократить время разработки моделей машинного обучения.



\---



\## Идея проекта



Пользователь загружает свой датасет и указывает цель анализа (например, классификация или регрессия).



Приложение автоматически выполняет первичный анализ данных:



\- определяет размер датасета;

\- извлекает список колонок;

\- определяет предполагаемые типы данных в колонках;

\- вычисляет основные статистические характеристики;

\- формирует небольшое превью содержимого датасета из случайных строк.



На основе этой информации создаётся компактное описание датасета в формате JSON.



\---



\## Использование нейросети



Сформированное описание датасета передаётся нейросети, которая выполняет анализ и генерирует рекомендации:



\- какие признаки стоит удалить;

\- какие признаки стоит преобразовать;

\- какие новые признаки можно создать;

\- какие модели машинного обучения можно попробовать.



\---



\## Результат



После анализа рекомендации структурируются и возвращаются пользователю в формате JSON.
